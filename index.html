Introduction

In order to view all of the samples that are included with this repo you will need to have the following open-source software installed on your operating system:

  • Node / npm

It is assumed that you have Node/npm (v6.5.0+) installed on your system, if not please refer to: Node/npm downloads.

HTML Sample

Open a Terminal window and execute the following commands:

  • Navigate to the "html" directory: cd html

  • Install npm modules: npm install

  • Start the http server: npm start

  • Open your browser and go to: http://localhost:3000

  • Run Jasmine tests:

    • With the web server running and while still in the "html" directory: npm test

    • All tests are located in the test directory

    • Test files are named: *.spec.js

Electron With Express Sample

Open a Terminal window and execute the following commands:

  • Navigate to the "electron-express" directory: cd html

  • Install npm modules: npm install

  • Start the electron application: npm start

  • Please note:

    • Since we are using Express within Electron, we need to spawn a child process:

                    //File: /samples/electron-express/app/main.js

              // Line 1
              // start electron, then start express
              const electron = require('electron'),
              spawn = require("child_process").spawn,
              exapp = spawn("node", ["./app/bin/www"]);

    • And then to correctly terminate the child process when the main window is closed:

                    //File: /samples/electron-express/app/main.js

              // Line 44
              if (process.platform !== 'darwin') {
                  // when we close the app, we need to kill the express server
                  process.kill(exapp.pid);
                  // quit main app
                  app.quit();
              }
